# LAB – VLAN e InterVLAN (Router-on-a-Stick)

Este laboratório tem como objetivo configurar VLANs, portas de acesso, VLAN padrão, trunk e conectividade InterVLAN utilizando Router-on-a-Stick.

## Objetivos do LAB

### A – Access Ports (RH, FINANCEIRO, ADMINS)
- Configurar portas de acesso para VLANs 10, 20 e 30.  
- Conectar corretamente cada host na VLAN correspondente.

### B – Default VLAN
- Verificar e entender o papel da VLAN 1 como VLAN padrão.  
- Remover portas da VLAN 1 e movê-las para suas VLANs específicas.

### C – InterVLAN Connectivity
- Criar subinterfaces no router (Router-on-a-Stick).  
- Testar conectividade entre VLANs via gateway (R1).
